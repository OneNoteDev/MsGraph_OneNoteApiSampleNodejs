API functionality demonstrated in this sample

The following aspects of the API are covered in this sample. You can find additional documentation at the links below.

  • Authenticate the user in an Node.js app
  • POST simple HTML to a new OneNote QuickNotes page
  • POST multi-part message with image data included in the request
  • POST page with a URL rendered as an image
  • POST page with a PDF file attachment

Prerequisites

Tools and Libraries you will need to download, install, and configure for your development environment.

  • Node.js
  • Express

Accounts

  • As the developer, you'll need to have a Microsoft account and get a client ID string so your app can authenticate with the Microsoft Live connect SDK.

Using the sample

After you've setup your development tools, and checked the prerequisites listed above,....

  1. Open a terminal and change directory to the project's directory.
  2. Setup project dependencies with the command: npm install
  3. Edit the config.js file to add in your Live Connect SDK information: Client ID, secret, redirect callback
  4. Run the app with the command: npm start
  5. Open a browser and navigate to the app running by default on port 3000.
  6. Authenticate in the running app, using your Microsoft account.
  7. Allow the app to create new pages in OneNote.
